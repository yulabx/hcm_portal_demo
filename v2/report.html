關鍵成果展示：統一管理介面

Single Pane of Glass (容器管理功能)

這項成果直接解決了用戶端複雜操作的問題，將底層 K8s 與 VM 資源抽象化，專注於「專案 Workload」維運。

  • 專案資源隔離：： 用戶登入即自動篩選其所屬專案的資源，確保權限隔離。
  • 即時狀態追蹤：： 工作負載狀態、資源配額消耗一目瞭然。
  • 一鍵維運操作：： 支援 Log 檢視與 Pod/VM 重啟，無需 SSH 登入。

AWS 雲端整合規劃 (最高優先級)

隔離實作：獨立 VPC

原則： 一個專案 = 一個獨立 VPC。

提供最嚴格的網路層 (L3) 隔離。將優先建立測試 VPC，驗證跨專案的網路阻斷策略。

資源識別：統一標籤

原則：： 全面套用 Tag: proj=xxx。

這是「雲地混合」的關鍵。透過一致的 Tag，將 AWS EC2 與地端資源視為同一資產管理。
